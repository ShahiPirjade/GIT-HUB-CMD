Initializes a new Git repository in the current directory.

	Commit history
	Branches
	Configuration
	Staging information

$git config --list
Displays all Git configuration settings.

$git status
It shows the current state of the working directory and staging area, including untracked, modified, and staged files.
Shows:
Current branch
Modified files
Staged files
Untracked files

$git add file1.txt
File is in the staging area.

$git commit -m "Added file1"

$git commit -am "Updated application code"
-a = Automatically stage all tracked files that have been modified or deleted. //i.e cannot use for initial file which is untracked. 
-m = Commit message
