###### To Configure Git Remote Repo and push it #######
$ mkdir myproject
$ cd myproject
$ git init

Step 1: Create Repository on GitHub
GitHub → New Repository [https://github.com/ShahiPirjade/GIT-HUB-CMD.git]

Step 2: Check Current Remote
git remote -v

Step 3: Add Remote Repository
git remote add <name> <repo-url>
$ git remote add origin https://github.com/ShahiPirjade/GIT-HUB-CMD.git
origin = alias name
origin is the default alias name given to a remote repository.

Change Existing Remote URL
$ git remote set-url origin https://github.com/user/newrepo.git

Remove Remote
$ git remote remove origin

Step 4: Verify Remote
git remote -v

Step 5: Push Code
$git push -u origin main
-u = upstream tracking
origin = remote name
main = branch name
# Do not work on master/ main branch 
eg git push -u origin
